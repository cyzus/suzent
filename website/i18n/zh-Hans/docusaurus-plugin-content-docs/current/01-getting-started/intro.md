# Suzent 是什么？

Suzent 是一款**主权本地智能体**：一个由你掌控、在本地优先运行的 AI 系统。无云端绑定；除非你主动选择，否则数据不会离开你的机器。

## 核心理念

**主权（Sovereign）** 意味着关键控制权留在你手里：模型密钥、数据、记忆和运行环境都由你决定。

**本地智能体（Local agent）** 意味着它不只是一次性回答问题。它可以保留长期记忆、安排定时任务，并在后台持续执行你允许的操作。

## 与众不同之处

| 特性 | Suzent |
|---|---|
| 模型 | 自带（GPT、Claude、Gemini 等均可） |
| 记忆 | 跨会话持久化——Markdown + 语义搜索 |
| 自动化 | 内置定时任务（Cron）和心跳监控 |
| 存储 | 本地优先，沙箱隔离执行 |
| 可扩展性 | 基于技能（Skills）的领域知识模块系统 |
| 连接性 | 通过节点（Nodes）以 WebSocket 连接伴侣设备 |

## 整体架构

```
你 ──► Suzent 智能体
             │
             ├── 你选择的 LLM（API 密钥）
             ├── 记忆系统（Markdown + LanceDB）
             ├── 工具（文件 I/O、网络、Bash、社交）
             ├── 技能（领域知识模块）
             ├── 自动化（Cron + 心跳）
             └── 节点（伴侣设备）
```
