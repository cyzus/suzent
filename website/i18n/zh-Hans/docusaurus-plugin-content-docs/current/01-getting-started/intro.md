# Suzent 是什么？

Suzent 是一款**超级个人协作智能体**——一个完全由你掌控的本地优先 AI 智能体。无云端绑定，除非你主动选择，否则数据不会离开你的机器。

## 核心理念

**主权（Sovereign）** 意味着所有事物由你掌控。你的模型密钥、你的数据、你的基础设施。Suzent 不会向外传送任何信息。

**数字协作者（Digital coworker）** 意味着它会随着时间与你一起工作——而不仅仅是回答一次性问题。它能记住过去的对话、安排定期任务，并在你休息时持续监控各类事务。

## 与众不同之处

| 特性 | Suzent |
|---|---|
| 模型 | 自带（GPT、Claude、Gemini 等均可） |
| 记忆 | 跨会话持久化——Markdown + 语义搜索 |
| 自动化 | 内置定时任务（Cron）和心跳监控 |
| 存储 | 本地优先，沙箱隔离执行 |
| 可扩展性 | 基于技能（Skills）的领域知识模块系统 |
| 连接性 | 通过节点（Nodes）以 WebSocket 连接伴侣设备 |

## 整体架构

```
你 ──► Suzent 智能体
             │
             ├── 你选择的 LLM（API 密钥）
             ├── 记忆系统（Markdown + LanceDB）
             ├── 工具（文件 I/O、网络、Bash、社交）
             ├── 技能（领域知识模块）
             ├── 自动化（Cron + 心跳）
             └── 节点（伴侣设备）
```
