超级个人协作智能体

SUZENT

掌控你的工作流。守护你的数据。按你的方式运行 AI。

快速开始 — 5 分钟在 GitHub 上查看

与模型无关

通过统一界面使用 GPT、Claude、Gemini 等。切换模型无需修改任何配置。

持久记忆

双层记忆架构：快速 Markdown 召回 + LanceDB 语义搜索。智能体跨会话记住一切。

默认私密

本地优先架构，沙箱隔离执行。你的数据始终留在你的机器上。

内置自动化

通过定时任务（Cron）安排任务，或运行持续的心跳监控——无需额外工具。

可通过技能扩展

将领域知识打包为 Markdown 技能。放入即用，智能体立刻获得新能力。

跨平台

支持 Windows、macOS 和 Linux。通过节点（Nodes）以 WebSocket 连接伴侣设备。

工作原理

01

安装

运行对应平台的安装脚本。Suzent 在一分钟内完成配置。

02

配置

打开 设置 → 提供商，添加你的模型 API 密钥。支持 GPT、Claude、Gemini 等任意兼容模型。

03

启动

运行 suzent start，立即上线——记忆、工具和自动化全部就绪。

5 分钟上手

安装 Suzent，配置你的模型密钥，启动你的第一个本地智能体会话。
