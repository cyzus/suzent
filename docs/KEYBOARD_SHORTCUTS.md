# Keyboard Shortcuts & Tips

Quick reference for keyboard shortcuts and productivity tips in Suzent.

## Chat Window

### Navigation
- `Ctrl/Cmd + N` - New chat
- `Ctrl/Cmd + K` - Focus search (chat list)
- `Ctrl/Cmd + ,` - Open settings
- `Escape` - Close sidebar/modal

### Message Actions
- `Enter` - Send message
- `Shift + Enter` - New line in message
- `Ctrl/Cmd + Enter` - Send message (alternative)
- `Ctrl/Cmd + /` - Stop current response

### Text Editing
- `Ctrl/Cmd + A` - Select all text
- `Ctrl/Cmd + C` - Copy selected text
- `Ctrl/Cmd + V` - Paste
- `Ctrl/Cmd + Z` - Undo
- `Ctrl/Cmd + Shift + Z` - Redo

## Sidebar

### Chat List
- `↑` / `↓` - Navigate chat list
- `Enter` - Open selected chat
- `Delete` - Delete selected chat (with confirmation)
- `Ctrl/Cmd + D` - Duplicate chat

### Plan View
- `Tab` - Cycle through task statuses
- `Space` - Toggle task completion
- `E` - Edit task

## Code Blocks

### Actions
- `Ctrl/Cmd + C` - Copy code (when hovering)
- Double-click - Select entire code block

## File Upload

### Images
- Drag & drop image into chat window
- `Ctrl/Cmd + V` - Paste image from clipboard
- Click image icon to browse files

## Browser Features

### General
- `Ctrl/Cmd + F` - Find in page
- `Ctrl/Cmd + R` - Refresh page
- `Ctrl/Cmd + +` - Zoom in
- `Ctrl/Cmd + -` - Zoom out
- `Ctrl/Cmd + 0` - Reset zoom

## Productivity Tips

### Quick Chat Start
1. Open Suzent
2. Start typing immediately (focus is on message input)
3. Press `Enter` to send

### Multi-line Messages
For formatted messages:
```
Use Shift + Enter to create multi-line messages
Perfect for:
- Lists
- Code snippets
- Structured questions
```

### Image Uploads
**Drag & Drop**: Fastest way to add images
1. Open file explorer
2. Drag image to chat window
3. Automatic upload and resize

**Paste from Clipboard**:
1. Copy image (screenshot, browser, etc.)
2. `Ctrl/Cmd + V` in message input
3. Image appears inline

### Code Block Selection
**Select code efficiently**:
1. Triple-click inside code block
2. Or click copy button (appears on hover)
3. Code is copied with proper formatting

### Search Chats
**Find chats quickly**:
1. Press `Ctrl/Cmd + K`
2. Type search query
3. Results filter in real-time
4. Press `Enter` to open first result

### Plan View
**Monitor task progress**:
1. Click "Plan" tab in sidebar
2. See all tasks and status
3. Click task to update status
4. Agent updates plan automatically

## Markdown Formatting

Suzent supports full Markdown in messages:

### Text Formatting
```markdown
**bold** or __bold__
*italic* or _italic_
***bold italic***
~~strikethrough~~
`inline code`
```

### Lists
```markdown
- Bullet point
- Another point
  - Nested point

1. Numbered list
2. Second item
   1. Nested number
```

### Links and Images
```markdown
[Link text](https://example.com)
![Alt text](image-url.png)
```

### Code Blocks
````markdown
```python
def hello():
    print("Hello, world!")
```
````

### Tables
```markdown
| Header 1 | Header 2 |
|----------|----------|
| Cell 1   | Cell 2   |
| Cell 3   | Cell 4   |
```

### Blockquotes
```markdown
> This is a quote
> Multiple lines
```

## Advanced Features

### Streaming Control
- **Stop response**: Click stop button or `Ctrl/Cmd + /`
- **Resume**: Not currently supported (will be added)

### Message Regeneration
1. Hover over user message
2. Click resend icon
3. Conversation truncated and regenerated

### Memory System (Optional)
- **View memories**: Click "Memory" tab
- **Edit core memory**: Click edit icon on memory blocks
- **Search archival**: Use search box in memory view

### Export/Import
- **Export chat**: Right-click chat → Export
- **Export all**: Settings → Export all chats
- **Import**: Settings → Import chat

## Configuration

### Model Selection
1. Click settings icon
2. Select "Config" tab
3. Choose model from dropdown
4. Changes apply to new messages

### Tool Selection
1. Open config view
2. Toggle tools on/off
3. Available tools:
   - Web Search
   - Planning
   - Web Page Fetch
   - File Operations
   - Memory (if enabled)

## Troubleshooting

### Keyboard Shortcuts Not Working
- Check if input is focused
- Try clicking elsewhere first
- Browser extensions may interfere

### Copy/Paste Issues
- Use browser copy/paste if app shortcuts fail
- Right-click → Copy as fallback
- Check clipboard permissions

### Slow Response
- Check network connection
- Stop and restart response
- Try different model

## Tips for Power Users

1. **Use keyboard exclusively**: Never touch mouse
   - `Ctrl+N` → Type → `Enter` → Read → Repeat

2. **Quick search**: `Ctrl+K` to find any past chat instantly

3. **Multi-tasking**: Open multiple browser tabs for parallel chats

4. **Export regularly**: Backup chats weekly with export all

5. **Template messages**: Create reusable prompts in separate file

6. **Code blocks**: Use ` ```language ` for syntax highlighting

7. **Planning**: Let agent create plan first for complex tasks

8. **Memory**: Update persona/user blocks for better context
