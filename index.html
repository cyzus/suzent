Get Started — 5 min

Model-agnostic

Use GPT, Claude, Gemini, and more with one interface. Switch models without rewriting your setup.

Persistent memory

Dual-tier memory: fast markdown recall + semantic LanceDB search. Your agent remembers across sessions.

Private by default

Local-first architecture with sandboxed execution. Your data stays on your machine.

Built-in automation

Schedule tasks with cron jobs or run continuous heartbeat monitors — no extra tooling needed.

Extensible via Skills

Package domain knowledge as Markdown skills. Drop them in and your agent instantly gains new capabilities.

Cross-platform

Works on Windows, macOS, and Linux. Connect companion devices via Nodes over WebSocket.
